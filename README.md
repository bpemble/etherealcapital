# Ethereal Capital

Corporate website for Ethereal Capital — a systematic proprietary trading and market-making firm specializing in CME futures options.

## Structure

```
index.html      — Single-page site
css/styles.css  — Stylesheet
js/main.js      — Animations and interactivity
assets/         — Favicon
media/          — Hero image
```

## Development

Static site with no build step. Open `index.html` in a browser or serve locally:

```sh
python3 -m http.server 8000
```

## Stack

- Vanilla HTML, CSS, JavaScript
- Google Fonts (Cormorant Garamond, Inter)
